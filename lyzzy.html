Lyzzy

A 2D platformer that makes use of physics, where the main character, a lizard, must use their tail for solving puzzles.

Developed in C# Unity for Windows and Playstation.

Contributions:

  • Character movement, controls and animations
  • Droping tail mechanic
  • Physics handling
  • Puzzle and level design implementation
  • Anti-softlock systems
  • UI implementation
  • Sound implementation
